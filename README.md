# 🌍 Landing Page - Agência de Viagens (Projeto Acadêmico)

## 📌 Sobre o Projeto

Este projeto consiste no desenvolvimento de uma **landing page para uma agência de viagens fictícia**, criada como parte de um trabalho acadêmico em equipe.

O objetivo é aplicar conceitos de desenvolvimento web, como:

* Estruturação com HTML
* Estilização com CSS
* (Opcional) Interatividade com JavaScript
* Boas práticas de UI/UX

---

## 🚀 Funcionalidades

* Apresentação da agência
* Destaque de destinos turísticos
* Sessão de pacotes/promos
* Formulário de contato
* Layout responsivo (mobile-first)

---

## 🛠️ Tecnologias Utilizadas

* HTML5
* CSS3
* JavaScript (se aplicável)
* Git & GitHub
